# This README is for CMT personal website Backend.

## Check me out at ...

* Ruby version 2.2.0

* Database is <tt>postgresql<tt>

* The test suite is RSpec, and can be run with <tt>rspec spec<tt>

* Deployed with Heroku

Please feel free to use a different markup language if you do not plan to run
<tt>rake doc:app</tt>.
